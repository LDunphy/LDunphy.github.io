Year 11 Topics

1. Programming fundamentals Explore
  • Syntax, variables, data types, and control structures.
  • Functions and modular programming.
  • Effective problem-solving and algorithms.
2. The object-oriented paradigm Explore
  • Concepts of classes, objects, and encapsulation.
  • Inheritance and polymorphism.
  • Applying OOP to solve complex problems.
3. Programming mechatronics Explore
  • Interfacing with hardware and sensors.
  • Control systems and robotics.
  • Developing software for mechanical systems.

Year 12 Topics

1. Secure software architecture Explore
  • Principles of secure design (e.g., least privilege, defense in depth).
  • Common vulnerabilities and threat modeling.
  • Defensive coding practices.
2. Programming for the web Explore
  • Front-end and back-end development.
  • Web frameworks and APIs.
  • Databases and data storage.
3. Software automation Explore
  • Continuous integration and continuous delivery (CI/CD).
  • Automated testing and deployment.
  • Scripting for routine tasks.
4. Software engineering project Explore
  • Managing a complete software development life cycle.
  • Documentation and user interface design.
  • Team collaboration and project management.
